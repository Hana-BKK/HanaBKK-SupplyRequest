เจ้าหน้าที่คลังพัสดุอนุมัติ

ระบบจัดการคลังพัสดุ Hana BKK

สามารถเพิ่ม ลด หรือแก้ไขรายการสินค้าในคลังสินค้าหลักได้

การตั้งค่าเชื่อมโยง Google Sheets

กรอกลิ้งก์ Webhook หรือ Google Apps Script เพื่อให้ระบบส่งข้อมูลไปลงชีตของคุณจริงๆ

คัดลอก Webhook URL ที่ได้จาก Google Apps Script (doPost) มาวางเพื่อส่งข้อมูลสั่งซื้อและเบิกแบบ Real-time

วิธีการตั้งค่ากับ Google Sheets

  1. เปิด Google Sheet ของคุณขึ้นมา
  2. ไปที่เมนู Extensions (ส่วนขยาย) > Apps Script
  3. คัดลอกและวางโค้ด Apps Script (สามารถดูคู่มือแนะนำสคริปต์ได้โดยคลิกด้านล่าง)
  4. ทำการ Deploy (ใช้งานได้) > New deployment (การทำให้ใช้งานได้ใหม่) เลือกประเภทเป็น Web App
  5. กำหนดสิทธิ์การเข้าถึงให้เป็น "Anyone" (ทุกคน) และนำ URL ที่ได้มาวางด้านบนนี้
